## Technologies Used
- Next.js
- Tailwind
- Supabase
- Shadcn.ui

## Features
- Visitor
- User
- Admin

## Installation
1. Clone the repository
```bash
git clone
```
2. Install dependencies
```bash
npm install
```
3. Start the server
```bash
npm start
```
