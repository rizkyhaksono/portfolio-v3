## Autoscaling

<LearningObjectives objectives={["Configure HorizontalPodAutoscaler", "Set CPU/memory requests and limits", "Use VPA for right-sizing", "Scale nodes with cluster autoscaler"]} />

```yaml
apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: api-hpa
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: api
  minReplicas: 2
  maxReplicas: 10
  metrics:
    - type: Resource
      resource:
        name: cpu
        target:
          type: Utilization
          averageUtilization: 70
```
