## GitOps with ArgoCD

<LearningObjectives objectives={["Define ArgoCD Application manifests", "Sync cluster state from Git", "Promote across environments", "Rollback with Git revert"]} />

```yaml
apiVersion: argoproj.io/v1alpha1
kind: Application
metadata:
  name: myapp
  namespace: argocd
spec:
  project: default
  source:
    repoURL: https://github.com/org/k8s-manifests
    path: apps/myapp
    targetRevision: main
  destination:
    server: https://kubernetes.default.svc
    namespace: production
  syncPolicy:
    automated:
      prune: true
      selfHeal: true
```
