<LearningObjectives objectives={["Write Pod and Deployment manifests", "Expose apps with ClusterIP and LoadBalancer Services", "Understand cluster DNS", "Restrict traffic with NetworkPolicy"]} />

```yaml
apiVersion: v1
kind: Service
metadata:
  name: api-service
spec:
  selector:
    app: api
  ports:
    - port: 80
      targetPort: 3000
  type: ClusterIP
```
