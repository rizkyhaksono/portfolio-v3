## Technical Details

### Architecture Improvements

We've completely refactored the core architecture:

```typescript
// Before
export function fetchData(id: string) {
  return fetch(`/api/data/${id}`).then(res => res.json())
}

// After
export async function fetchData(id: string): Promise<Data> {
  const response = await fetch(`/api/v2/data/${id}`)
  if (!response.ok) throw new Error('Failed to fetch')
  return response.json()
}
```
